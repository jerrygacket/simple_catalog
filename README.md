# Products catalog
Simple products catalog with quick filtering. 
Contains 
- two go-lang microservices, 
- one PHP app for backend and db deploy, 
- one Vue app for frontend.
## Languages
- PHP
- SQL
- JavaScript
- HTML
- CSS
- GO
## Technologies
- Docker
- Nginx
- PHP-FPM
- Reindexer
- microservices
## Build & Deploy
```shell
make install
make
```
