# Products catalog
Simple products catalog with quick filtering. 
Contains 
- two go-lang microservices, 
- one PHP app for backend and db deploy, 
- one Vue app for frontend.
## Languages
- PHP
- SQL
- JavaScript
- HTML
- CSS
- GO
## Technologies
- Docker
- Nginx
- Percona (SQL)
- PHP-FPM
- Reindexer (NoSQL)
- microservices
## Build & Deploy
```shell
git clone https://github.com/jerrygacket/simple_catalog.git
cd simple_catalog
make all
```
